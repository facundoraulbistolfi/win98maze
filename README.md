# Win98 Maze

**Laberinto 3D en primera persona con estética Windows 98**

[![GitHub Pages](https://img.shields.io/badge/Jugar-GitHub%20Pages-0072C6?style=for-the-badge&logo=github)](https://facundoraulbistolfi.github.io/win98maze)

</div>

---

## De qué se trata

Un laberinto 3D en primera persona. El objetivo es encontrar la rata. Si elegís las texturas **Backrooms** en paredes, piso y techo, aparece el monstruo.

---

## Controles

```
W / S          — Avanzar / Retroceder
A / D          — Strafe izquierda / derecha
Flechas        — Girar
Tab            — Ciclar minimapa
M              — Menú de configuración
```

En mobile hay dos joysticks virtuales.

---

## Configuración (menú M)

- **Texturas**: 6 opciones para paredes, piso y techo (Ladrillo, Piedra, Metal, Madera, Azulejo, Backrooms)
- **Minimapa**: posición (4 esquinas o centro rotante) y opacidad
- Seleccionar Backrooms en los tres slots activa el monstruo que te persigue por el laberinto

---

## Stack

- React + Three.js
- Generación de laberintos con Recursive Backtracker
- Pathfinding del monstruo con BFS
